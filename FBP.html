SECRETÁRIA DE EDUCAÇÃO PROFISSIONAL E TECNOLOGIA

Fundamentos e Projeto de Banco de Dados

Carga horária

Carga horária total: 72 horas.

Ementa

Conhecimentos básicos sobre Bancos de Dados e Sistemas de Gerenciamento de Bancos de Dados; Aspectos de projeto; Acesso a dados e linguagem de manipulação de dados; Modelos de dados e sua evolução: modelo hierárquico, modelo de rede e modelo relacional.

Competências

  1. Conhecer os conceitos de banco de dados (BD)
  2. Compreender os conceitos de sistema de gerência de banco de dados (SGBD);
  3. Caracterizar e conhecer as vantagens de um SGBD.

Habilidades

  1. Diferenciação dos sistemas de armazenamento de dados;
  2. Diferenciação das formas de modelagem de banco de dados;
  3. Realização de modelagem de banco de dados a partir de um estudo de caso
  4. Preparação de modelo lógico consistente;
  5. Transformação dos modelos conceituais em lógicos;
  6. Desenvolvimento do modelo lógico e físico;
  7. Desenvolvimento de normalização de relações;
  8. Desenvolvimento da Álgebra Relacional para consultas procedurais;
  9. Realização de manipulações básicas de dados através de linguagem SQL.
